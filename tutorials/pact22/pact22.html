COMET Compiler

Write once, run everywhere...

PACT'22 -- October 9th, 2022

Introduction to COMET

Logistics: Sunday October 9th, 2022 -- 9.00am-12.00pm CDT

Objectives: Write sparse and dense tensor algebra kernels in COMET DSL, Apply various code optimizations and transformations for high performance, and FPGA code generation flow

Abstract: This tutorial will provide an overview of the COMET compiler and its supported frontends. It will also review the optimizations and transformations supported by the COMET compiler for efficient code generation on target architectures. At the end of this tutorial, attendees should learn how to write various dense and sparse tensor algebra kernels using the COMET DSL, perform sophisticated code optimizations and transformations
